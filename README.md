# brainSurfer
brainSurfer is a toolbox for visualizing surface space data, or for projecting volume space data onto surface space.

Plop all of these files in any directory you want and add then to your path in matlab (with subdirectories). Edit stuff at your own peril! Type 'brainSurfer' into matlab to display GUI. Use plotUnderlay and plotOverlay to write your own scripts. Hover over buttons in the GUI to understand what they do. Report bugs to alex.teghipco@uci.edu! This software is for experimental use only, and uses some other toolboxes (all provided in the ./scripts folder).
